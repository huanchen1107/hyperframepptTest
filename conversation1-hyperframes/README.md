# SMC x DRL Storyboard-Driven HyperFrames Video

This HyperFrames composition converts a 16-slide SMC and deep reinforcement
learning presentation into a narrated, captioned, animated video.

## Included

- 16 rendered slide assets
- Taiwan Mandarin professional male narration
- Synchronized animated captions
- Per-slide visual object focus paths
- HUD, scanning, camera motion, and section transitions
- Storyboard-driven animation choreography

## Commands

```powershell
npm run dev
npm run check
npm run render
```

The composition is defined in `index.html`. The rendered final MP4 and detailed
slide-by-slide storyboard are available in the repository's `deliverables/`
directory.
