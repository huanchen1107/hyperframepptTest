# Introduction to Open Design

Welcome to the presentation on Open Design. Today we will cover the core concepts and capabilities.

## What is Open Design?

- Local-first, open-source design tool.
- Native desktop app for macOS and Windows.
- Runs entirely on local coding agents.
- Protects privacy by keeping data on your laptop.

## Core Features

- **Prototypes**: Web, desktop, and mobile HTML artifacts.
- **HyperFrames**: Programmatic motion and animated graphics.
- **Decks**: Slide presentations exported to PDF or PPTX.
- **Images**: High-resolution brand-grade visual assets.

## The Workflow

- Start with a brief.
- Lock the design direction.
- Stream the artifact in real-time.
- Hand off to engineering effortlessly.

## Conclusion

Open Design is the Figma alternative for the agent era. Let's start building!
